# Lumosyn Energy Estimate

Milestone 1 creates the production-ready foundation for the Lumosyn Energy Estimate app.

This milestone includes:

- Next.js App Router with TypeScript
- Tailwind CSS
- ESLint
- Lucide React dependency
- PWA configuration with `next-pwa`
- Placeholder landing page

This milestone intentionally excludes business logic, API routes, authentication, Supabase, WhatsApp, forms, analytics, and database work.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Deployment

The app is ready for Vercel deployment without environment variables.
