# Lumosyn Analytics

Vercel Web Analytics shows visits, page views, and referrers in Vercel.

Vercel Speed Insights helps monitor real-user performance.

Supabase assessments show which campaign completed an estimate.

No cookies, IP storage, or device fingerprinting are being added.

For now, success means:

- A visitor lands from a tracked link.
- The estimate is completed.
- The assessment row saves with UTM data.

## Post-Deploy Step

In the Vercel dashboard:

- Enable Web Analytics for the project.
- Enable Speed Insights for the project.
- After deployment and a few visits, check the Analytics and Speed Insights dashboards.
